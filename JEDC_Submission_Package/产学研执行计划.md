# TFI 项目：长周期产学研与在职创业执行计划 (6年)

**核心目标**：以北京大学非全日制硕士身份起步，通过 **[副业变现 + 天使融资]** 完成资本积累，**带资进组**攻读博士，最终实现 **[博士毕业 + 公司成熟]** 的双重自由。
**当前状态**：技术闭环（论文/代码/Demo已就绪），负债 30w，在职（月薪 2w，暂不跳槽）。
**战略定调**：**在职创业 (Side Hustle to Startup)** —— 利用本职工作保底，利用副业造血，利用学术成果撬动天使融资。

---

## 第一阶段：技术资产化与冷启动 (Year 1: The Hustle)
**目标**：完成技术确权，构建**“一鱼多吃”的论文矩阵**，副业月入过万。

*   **Month 1: 软著与核心投稿**
    *   [ ] **软著申请**：个人名义申请 TFI 系统软著（确保 IP 归属）。
    *   [ ] **JEDC 投稿 (The Flagship)**：投递金融论文，作为融资和申博的背书。
    *   [ ] **arXiv 占坑 (The Shield)**：上传《智能的几何》，抢占“Transformer 物理学解释”首发权。

*   **Month 2-3: 衍生论文与演示系统**
    *   [ ] **论文 C (Crypto)**：《基于拓扑序参量的加密货币市场崩盘预警》，投递 **Finance Research Letters** (SSCI 一区)。
    *   [ ] **论文 E (Emerging Markets)**：《中国 A 股市场的微观结构相变》，投递 **Pacific-Basin Finance Journal**。
    *   [ ] **Demo 上线**：将 `demo/app.py` 部署到云服务器（阿里云/Streamlit Cloud），生成可访问的 URL，作为“产品名片”。

*   **Month 3-6: 建立影响力 (IP Building)**
    *   [ ] **自媒体矩阵**：
        *   **知乎/公众号**：定位“硬核量化风控”，连载《系统科学金融理论》系列文章（基于您的学术综述）。
        *   **小红书**：发布 TFI 信号图，打标签 #北大博士 #量化交易 #A股逃顶。
    *   [ ] **私域沉淀**：所有流量引导至个人微信/企微，建立“TFI 研讨群”。

*   **Month 7-12: 副业变现 (Cash Flow - The "Robot" Strategy)**
    *   **核心逻辑**：利用“5:30下班”的时间红利，将 Demo 升级为全自动信号源，实现睡后收入。
    *   [ ] **产品化 (Signal Bot)**：
        *   在 `app.py` 后台部署定时任务（Cronjob）。
        *   开发自动推送脚本，将 TFI 日报推送到钉钉群/飞书群/微信群。
    *   [ ] **阶梯式收割**：
        *   **L1 (引流)**：知乎/小红书自动发布 TFI 图表，吸粉进群。
        *   **L2 (订阅)**：TFI 信号群会员（¥365/年），机器人自动服务，零人工成本。目标 200 人（营收 7.3w）。
        *   **L3 (高客单)**：出售“顶刊复现源码包+视频讲解”（¥1999/单）。目标每月 2 单（营收 4.8w）。
    *   *财务目标*：副业年收入 12w+，配合工资结余，**优先构建现金储备（读博基金），而非提前还贷（利用学贷低息优势）**。

## 第二阶段：融资与读博准备 (Year 2: The Leverage)
**目标**：寻找天使投资，积累 20w+ 可支配现金，拿到博士录取通知书。

*   **Month 13-18: 天使融资 (Angel Round)**
    *   [ ] **BP 撰写**：基于 JEDC 论文和一年的实盘/副业数据，撰写商业计划书。
    *   [ ] **精准找钱**：
        *   **目标对象**：北大校友、副业积累的高净值粉丝、量化圈老板。
        *   **融资目标**：50w - 100w RMB（出让 5%-10% 股份）。
        *   *用途*：这笔钱不是用来烧的，是用来**置换您的时间**（还债 + 读博期间生活费 + 少量运营成本）。
    *   [ ] **成立公司**：拿到钱后，注册公司（您控股），正式从“副业”转为“创业”。

*   **Month 19-24: 博士申请 (The Upgrade)**
    *   [ ] **学术大杀器**：此时您手握 **[JEDC 录用 + 几何智能论文 + 运营中的公司]**。
    *   [ ] **联系博导**：寻找支持产学研转化的博导。明确提出“我自带项目和资金，读博是为了深化理论和平台背书”。
    *   [ ] **成功入学**：身份转换为“全日制博士创业者”。

## 第三阶段：博士创业期 (Year 3-6: The Scale-up)
**目标**：利用学校资源（补贴/人脉）放大业务，完成 TFI 2.0 研发。

*   **学术侧**：
    *   利用博士期间的算力资源，打磨《智能的几何》论文，冲击 Nature 子刊/NeurIPS。
    *   这将是公司下一轮（A轮）融资的核心技术壁垒。

*   **商业侧**：
    *   **申请补贴**：以博士身份申请高校创业补贴（几十万）。
    *   **产品迭代**：从单一的 TFI 指标升级为“AI 驱动的资管风控 SaaS”。
    *   **扩大规模**：招募实习生（学弟学妹），低成本扩张团队。

## 财务推演 (Financial Projection)

*   **Year 1**：工资 24w + 副业 10w = 34w。还债 15w，生活+运营 19w。剩余债务 15w。
*   **Year 2**：工资 24w + 副业 20w = 44w。还清剩余 15w，积蓄 20w。**无债一身轻**。
    *   *若融资成功*：额外获得 50w-100w 现金储备。
*   **Year 3-6 (读博)**：
    *   收入：公司分红 + 学校补贴 > 覆盖生活费。
    *   资产：持有公司高额股权（潜在价值千万）。

## 风险控制

1.  **合规红线**：副业变现严禁涉及“非法荐股”或“代客理财”。只做**技术服务**和**投资者教育**。
2.  **时间管理**：在职做副业极累。必须利用自动化工具（如脚本自动发信号）降低边际成本。
3.  **融资心态**：如果融不到资，不要强求。依靠副业现金流也能读博，只是稍微辛苦点。**不要为了融资签对赌协议。**
